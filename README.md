# Instructions

1. Sign-in to [enterprise.hackthebox.com](https://enterprise.hackthebox.com)
2. Retrieve cookie value `hack_the_box_enterprise_platform_session`
3. Go to [Netlify project settings > Environment variables](https://app.netlify.com/projects/htbzh/configuration/env#environment-variables)
4. Save value to `HTB_SESSION`
5. Trigger a [redeploy](https://app.netlify.com/projects/htbzh/deploys) on Netlify
